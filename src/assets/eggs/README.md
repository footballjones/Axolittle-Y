# Egg Artwork

Store egg artwork here. Suggested naming convention:

- `egg-common.png` — Common rarity egg
- `egg-uncommon.png` — Uncommon rarity egg
- `egg-rare.png` — Rare rarity egg
- `egg-legendary.png` — Legendary rarity egg
- `egg-default.png` — Fallback / generic egg (used in hatching intro)

Import example:
```ts
import eggCommon from '../../assets/eggs/egg-common.png';
```
